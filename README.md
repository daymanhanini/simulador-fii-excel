# Simulador de Investimentos em Fundos Imobiliários (FII) no Excel

Projeto desenvolvido para o desafio da DIO com o objetivo de aplicar conceitos de Excel na criação de uma ferramenta prática de simulação de investimentos em Fundos Imobiliários (FIIs).

A planilha permite simular aportes mensais, tempo de investimento, rendimento mensal e estimativa de dividendos, mostrando a evolução do patrimônio mês a mês.

---

## Funcionalidades

- Simulação com aporte inicial e aporte mensal
- Cálculo do patrimônio acumulado com rendimento mensal
- Estimativa de dividendos mensais
- Evolução mês a mês em tabela
- Gráfico de evolução do patrimônio

---

## Como usar

1. Abra o arquivo `DIO Desafio.xlsx`
2. Vá até a aba **simulador**
3. Preencha os campos:
   - Aporte inicial
   - Aporte mensal
   - Tempo (meses)
   - Rendimento mensal (%)
   - Taxa de dividendos (%)
4. Confira os resultados na aba **simulador**
5. Veja a evolução completa na aba **evolução**
